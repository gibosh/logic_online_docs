# Logic+ Online

Logic+ Online is a web-based schedule analysis tool. Upload construction project schedules and use Logic+ to understand schedule health, delay risk, and critical path behaviour. 

---

New to Logic+? [Getting Started](guides/getting-started) walks through the upload workflow and what to expect from each module. 

See the [Glossary](guides/glossary) for key terms.

---
